# 学习路线

从浅入深逐步理解 cc-python-claude 的架构。

## 路线图

| 步骤 | 主题 | 核心文件 | 目标 |
|------|------|---------|------|
| 1 | 跑起来 | `cc/__main__.py` | 建立直觉：这是一个 tool-use agent，不是 chatbot |
| 2 | 入口与装配 | `cc/main.py` | 理解模块组装和 REPL 驱动循环 |
| 3 | 数据模型 | `cc/models/content_blocks.py`, `cc/models/messages.py` | 理解 agent 和 API 之间"说什么话" |
| 4 | 事件系统 | `cc/core/events.py` | 理解 query_loop 和 UI 之间的"神经系统" |
| 5 | 状态机内核 | `cc/core/query_loop.py` | 理解 agent 的"心脏"：while(true) 循环 |
| 6 | 调用方与依赖 | `cc/core/query_engine.py`, `cc/api/claude.py`, `cc/tools/streaming_executor.py` | 理解完整调用链 |

## 读代码的方法

1. 先读每个文件顶部的模块文档注释（这个项目的注释质量很高）
2. 再看代码，遇到不懂的引用再跳过去读
3. 每一步都是下一步的铺垫，不要跳步
